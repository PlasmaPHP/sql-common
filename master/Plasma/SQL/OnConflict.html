class OnConflict (View source)

Represents an ON CONFLICT resolution.

Constants

RESOLUTION_DO_NOTHING

On conflict, nothing will be done. The error will be ignred.

const RESOLUTION_DO_NOTHING = 1;
RESOLUTION_ERROR

On conflict, an error will be raised. This it the default.

const RESOLUTION_ERROR = 0;
RESOLUTION_REPLACE_ALL

On conflict, replaces all columns, except the primary key.

This includes auto generated columns such as created_at and updated_at.

const RESOLUTION_REPLACE_ALL = 2;
RESOLUTION_REPLACE_COLUMNS

On conflict, it will replace the defined columns.

const RESOLUTION_REPLACE_COLUMNS = 4;
